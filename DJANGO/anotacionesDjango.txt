CREAR UN NUEVO PROYECTO CON DJANGO


1-------PASO:
en cmd ->	django-admin startproject nombre-proyecto
	LUEGO SE CREA

		manage.py ->  ARCHIVO IMPORTANTE
		creado y 
		carpeta con archivos -> 	* __init_.py : es importante para que python trate el proyecto como un paquete
						* setting.py : contiene las configuraciones del proyecto de python
						* urls.py : se almacenan las urls que va a usar nuestro proyecto
						* wwsgi.py : para el servidor que vamos a usar

2-------PASO: 
	hacer conexion con la base de datos , de momento sqllite
	en la consola te diriges al directorio
	escribes	

		python manage.py migrate
	

	luego aparecera la base de datos SQLITE3

3-------PASO: 	
	HACER TESTEO EN EL SERVIDOR DE DJANGO : > ojo en un entorno real no es recomendable usarlo ya que parece ser monolitico o de un solo hilo y no 
	soporta multiples usuarios ni transacciones
		
		python manage.py runserver

4------PASO:
	PARA CERRAR EL SERVIDOR ES LO MISMO QUE EN REACT JS O NODEJS 
	CTRL + CTRL + C
